# Region Resonances — React App

The user-facing tool. A Vinotheca companion that matches user-entered metaphors to wine regions via semantic similarity over pre-computed embeddings.

## What it does

1. Loads `region_embeddings.json` (1.8 MB, ships with the app) — 59 wine regions with their canonical metaphors, identity narratives, and 1536-dim OpenAI embeddings.
2. When the user types a query, debounces 400ms, then POSTs the query to a Cloudflare Worker that returns an embedding for it.
3. Computes cosine similarity between the query embedding and each region's embedding, ranks all 59, and renders the top matches.
4. When no query is active, displays browse mode — all 59 regions grouped by cluster, click-to-expand for the full narrative.

## Run locally

```bash
# Install deps
npm install

# Start dev server (opens browser at http://localhost:5173)
npm run dev
```

**Important:** before `npm run dev` will serve real data, you need to make `region_embeddings.json` available to the app. The file is generated by `../scripts/build_metaphor_embeddings.py` and lives in `../public/data/region_embeddings.json`. Vite serves from `app/public/` though, so we need to either:

- Copy: `cp ../public/data/region_embeddings.json public/region_embeddings.json` (simple, must repeat after each rebuild)
- Symlink: `ln -s ../../public/data/region_embeddings.json public/region_embeddings.json` (set once, follows updates)

The symlink is cleaner.

## Build for production

```bash
npm run build
```

Outputs to `dist/`. Deploy that directory to GitHub Pages.

## Visual language

Matches `region-affinities` exactly — same Tailwind config, same `nav-link` / `pill` / `small-caps` patterns, same EB Garamond + Inter pairing. Region Resonances additions are documented in `src/styles/index.css`:

- `.metaphor` — italic wine-red, used wherever the one-word metaphor appears
- `.pill--<cluster-class>` — cluster-specific pill colours
- `.card` / `.card-interactive` — soft parchment-warm container, subtle hover lift
- `.chip` — example query chips, parchment with wine hover
- `.resonance-input` — the main freeform input, large serif type
- `.drop-cap` — editorial drop-cap on expanded narratives
- `.fade-in-stagger` — staggered fade-in for cards as results stream

## Configuration

- **Worker URL** — hardcoded in `src/lib/api.js` as `https://region-resonances-proxy.jskarabot.workers.dev`. Update there if the Worker moves.
- **Region Affinities deep-link pattern** — in `src/lib/regionLinks.js`. Currently uses hash routing: `https://jskarabot18.github.io/region-affinities/#/atlas?region=<id>`. Update if Region Affinities changes its URL scheme.
- **Tier counts** — top 3 hero / 4-7 secondary / 8+ in expandable. Constants in `src/components/ResultsPanel.jsx` (`HERO_COUNT`, `SECONDARY_END`).
- **Debounce delay** — 400ms in `src/App.jsx` where `useDebouncedValue` is called.
